1. Installation
	1.1. Copy all the files and folders in the root directory
	1.2. Create a database
	1.3. In the file "classes/_class.config.php", fill in the configuration data
	1.4. Run "install_DB.php" file to automatically create tables in the database, or
		import the table from the folder "DB/data_of_urls.sql"
	1.5. Remove "install_DB.php" file

1. Инсталяция
	1.1. Скопируйте все файлы и папки в корневую директорию
	1.2. Создайте БД
	1.3. В файле "classes/_class.config.php", внесите конфигурационные данные
	1.4. Запустите файл "install_DB.php" для автоматического создания таблицы в БД или 
		импортируйте таблицу из папки "DB/data_of_urls.sql"
	1.5. Удалите файл "install_DB.php"
